# Infuse.js Dark

> Infuse.js Dark syntax theme

[![Demo of Infuse.js Dark syntax theme](./demo.png)](./demo.png)

All one has to do is clone this repo into the VS Code extensions directory.

On macOS, this is located in a hidden folder:

```console
~/.vscode/extensions
```

**Enjoy!**
